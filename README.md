# dotfiles

Config personal para Hyprland en Arch Linux.

## Contenido

| Directorio | Destino | Descripción |
|------------|---------|-------------|
| `config/hypr` | `~/.config/hypr` | Hyprland, hypridle, hyprlock |
| `config/waybar` | `~/.config/waybar` | Barra de estado |
| `config/rofi` | `~/.config/rofi` | Lanzador de apps |
| `config/kitty` | `~/.config/kitty` | Terminal |
| `config/nvim` | `~/.config/nvim` | Neovim |
| `config/dunst` | `~/.config/dunst` | Notificaciones |
| `config/Thunar` | `~/.config/Thunar` | Gestor de ficheros |
| `config/zsh/zshrc` | `~/.zshrc` | Shell |

## Instalación

```bash
git clone https://github.com/miuser/dotfiles ~/Repos/dotfiles
cd ~/Repos/dotfiles
bash install.sh
```
